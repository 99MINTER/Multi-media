1. Define multimedia and list five elements that make up a multimedia system. expand_more

Multimedia refers to the integration of multiple forms of media—such as text, images, audio, video, and animations—into a single digital system to communicate information interactively or non-interactively.

Five elements of a multimedia system:

  • Text – Written information used for communication and instructions.
  • Images/Graphics – Photos, drawings, and illustrations that enhance visuals.
  • Audio – Sound, music, or voice narration.
  • Video – Moving visual media for demonstrations or storytelling.
  • Animation – Motion graphics that simulate movement or processes.
2. Differentiate between Analog Audio and Digital Audio. expand_more

Analog Audio

  • Continuous sound wave that varies smoothly over time.
  • Captures natural sound directly as electrical signals.
  • More prone to noise, distortion, and signal degradation.
  • Examples: Cassette tapes, vinyl records.

Digital Audio

  • Discrete sound represented in binary (0s and 1s).
  • Sound is sampled and converted into digital format.
  • High clarity, less noise, and easy to store or process.
  • Examples: MP3, WAV, digital recordings.
3. Differentiate between linear and non-linear multimedia projects with examples. expand_more

Linear Multimedia

  • Content flows in a predetermined sequence, without user control.
  • The user is a passive viewer.
  • Example: Movie played start to finish, a TV broadcast, an auto-running slideshow.

Non-Linear Multimedia

  • User can interact and control navigation—choose what to view and when.
  • Interactive interfaces.
  • Example: Websites, e-learning platforms, video games, interactive tutorials.
4. What is Hypertext and how does it differ from Hypermedia? expand_more

Hypertext

  • Text-based information linked through clickable hyperlinks.
  • Allows navigation between documents or sections.

Hypermedia

  • An extension of hypertext that includes multimedia elements (audio, images, videos, animations) as linked content.

Difference:

Hypertext involves only text links, while hypermedia integrates multimedia in addition to text.

5. Mention four basic tools used for multimedia development and their purposes. expand_more
  1. Graphics/Photo Editing Tools (e.g., Adobe Photoshop) – Create and edit images and graphics.
  2. Audio Editing Tools (e.g., Audacity) – Record, mix, and edit sound files.
  3. Video Editing Tools (e.g., Adobe Premiere Pro) – Edit, cut, and assemble video sequences.
  4. Authoring Tools (e.g., Adobe Animate, Unity, PowerPoint) – Integrate text, images, audio, and video into an interactive project.
6. Outline the components (hardware and software) required for a multimedia system. expand_more

Hardware Components

  • Computer system with high processing power
  • Sound card for audio input/output
  • Speakers/headphones
  • Microphone for audio recording
  • Digital camera/camcorder for capturing images or videos
  • Scanner (for graphics)
  • Large storage devices (SSD/HDD)
  • High-resolution display/monitor

Software Components

  • Operating system (Windows, macOS, Linux)
  • Graphics editing software
  • Audio & video editing applications
  • Multimedia authoring tools
  • Playback software (media players, codecs)
7. State four characteristics of a multimedia system. expand_more
  1. Interactivity – Users can control or influence content.
  2. Integration – Combines multiple media forms (text, audio, video, graphics).
  3. Digital representation – Uses digital formats for storage and processing.
  4. User engagement – Provides immersive and appealing content.
8. Identify and briefly explain any three challenges faced by multimedia systems. expand_more
  • High Storage Requirements
    Multimedia files (especially video and high-quality audio) consume large disk space.
  • High Processing Power Needed
    Running and rendering multimedia requires powerful CPUs and GPUs.
  • Compatibility Issues
    Different file formats, codecs, and software may not work smoothly across all systems.
9. What are the system implications of implementing a multimedia system? expand_more

Implementing a multimedia system affects:

  • Hardware Requirements: Need for faster processors, more RAM, better graphics cards, and large storage.
  • Software Demands: Need specialized editing, creation, and playback software.
  • Network Load: Large multimedia files require high bandwidth.
  • Cost Implications: Upgrading equipment and software licenses increases operating costs.
  • Maintenance Needs: Requires trained personnel and frequent system updates.
10. Describe any three file formats used in multimedia and their respective uses. expand_more
  1. MP3 (Audio Format)
    Compressed audio format used for music, podcasts, and web audio.
  2. MP4 (Video Format)
    Standard video format used for streaming, online video, mobile video playback.
  3. JPEG/JPG (Image Format)
    Compressed image format used for photographs and web images.

Other examples include: WAV, AVI, PNG, GIF, MOV, FLV.
